Variables

Perl variables come in 3 flavors.

Scalars

A scalar is a single value: a number, a string, or a reference. Scalars variables always start with a $. For example:

my $foo = 'dinosaur';

Lists

A list is an ordered set of scalar values. List variables always start with an @. For example:

my @foo = (1, 2, 3, 'cowboy');
print $foo[1]; # prints 2

Hashes

A hash is an unordered collection of key/value pairs. Hash variables always start with a %. For example:

my %foo = (a => 1, "b" => 2, 'c' => 'batman');
print $foo{c}; # prints 'batman'
